# YieldMind UI Preview

## Dark Terminal Aesthetic with Neon Lime Theme

The YieldMind dashboard features a distinctive dark terminal UI with neon lime (#39FF14) accents that create a cyberpunk, Matrix-like aesthetic perfect for a DeFi AI optimizer.

### Color Scheme
- **Background**: Deep black (#0a0a0a) - creates maximum contrast
- **Panels**: Dark gray (#1a1a1a) - subtle separation from background  
- **Primary Accent**: Neon lime (#39FF14) - high visibility, tech-forward feel
- **Borders**: Medium gray (#2a2a2a) - subtle element separation
- **Text**: Neon lime on black - excellent readability, terminal feel

### Visual Effects

**Pulsing Animations**
- AI status indicator pulses to show it's active
- Active protocol badge pulses to draw attention
- Subtle 2-second fade in/out cycle

**Glow Effects**
- All borders have subtle neon lime glow
- Vault panel has stronger glow (most important metric)
- Hover interactions increase glow intensity
- Text shadows create terminal CRT-like effect

**Transitions**
- Smooth 0.3s transitions on hover
- Border colors intensify on interaction
- Scale transforms on buttons (1.05x)

### Typography
- **Font**: Courier New (monospace) - classic terminal feel
- **Sizes**: 
  - Title: 2.5rem (40px)
  - Balance: 2rem (32px)
  - APY: 1.5rem (24px)
  - Body: 1rem (16px)
  - Timestamps: 0.8rem (13px)

### Responsive Design
- Grid collapses to 1 column on mobile
- Touch-friendly spacing
- Maintains readability at all sizes

### Theme Inspiration
- Matrix-style terminal interface
- Cyberpunk aesthetics
- Hacker/developer friendly
- High contrast for visibility
- Futuristic DeFi branding
